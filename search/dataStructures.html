A. Lists

Lists are dynamic (expands as much as needed), heterogeneous (different types of data can exist at the same list) arrays contain sequence of items

List creation:

  • myList = [] then add using append or insert
  • myList = [1, 2, 3, 4] filled with initial data
  • myList = list((1, 2, 3, 4))
Access, modify and delete from a list
  • Access by index / range of indexes (also used for slicing) / for each loop to access all elements-> myList[0:-2]
  • list.append() is used to add an element at the end of the list
  • list.insert() is used to add an element at a specific given index
  • list.remove() is used to remove a specific element from the list
Other common methods:
  • list.extend() is used to merge two lists together or we can use + to join two lists together
  • pop() is used to access the last element of the list by default or pop a specific given index
  • clear() will remove all elements from the list
  • index() will remove the index of the specified value
  • count() counts the number of the occurrences of the specified value
  • Check for items exits in the list if "a" in myList: print("yes")
  • len() returns the length of the list

list comprehensions and their advantages

It is away with shorter syntax (one line of code) when you want to make a new list based on the values of an existing list

It replaces the work of the for each loop and merge it into one line

Assuming we have a list myList = [1,2,3,4,5,6,7,8,9]
task: we want to make a new list that contain the odd numbers from the main list using list comprehension
newList = [x for x in myList if x%2 != 0
